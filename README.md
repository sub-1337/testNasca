# testNasca

## Биржевый калькулятор

### Сборка
Перейдите в папку calculator.

Поправьте cmake файл - вам нужно указать актуальный
путь до библиотеки boost, остальные библиотеки будут
скачаны автоматически. Это известная особенность 
библиотеки буста устроены немного иначе чем другие.

Создайте папку для выходных файлов. Запустите cmake.
`cmake -S . -B out/`
Далее постройте своим компилятором.
Работоспособность тестировалась на **VS 2022**

Проект разделйн на 3 проекта:

1) `calculator`
Исполняемый файл принимающий параметры.
2) `calculator_lib`
Статическая библиотека с ядром функционала.
3) `calculator_tests`
Тесты.

В целом функционал реализован близко к заданию.
